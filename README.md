# Flame
Flame is a minimalist game engine written in C++.
## Features
- ⚪️ Primitives [Square & Circle, Triangle coming soon!]
- 🏃‍♀️ Sprites
- 🔥 GPU Accelerated primitives*
- 🎮 Inputs
